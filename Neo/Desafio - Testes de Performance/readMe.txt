Com o Node instalado na máquina, executar o seguinte comando no prompt:

npx serverest@latest

-------------------------------------------------------------------------------

Para instalar o JMeter, é preciso ter primeiramente o Java instalado, após isso é possível baixá-lo no link:

https://jmeter.apache.org/download_jmeter.cgi

E abri-lo executando o arquivo "ApacheJMeter" dentro da pasta "bin"
